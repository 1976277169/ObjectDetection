###依赖项目

使用前请先下载opencv和pthreads-w32，同时对项目属性进行配置，项目属性的配置可以参考文件夹中的DPM_debug64_vc12.props和DPM_release64_vc12.props。

###图像数据库

本实验针对2014年智慧城市比赛数据库，如果想使用自己的图像，请针对main.cpp和xml_make.cpp作出相应改变。

###内存依赖性

使用者可以根据自己内存的情况更改Get_BG.cpp的Line 69 和Line 71中的scale值。
建议，4G内存：（8,4），16G及以上内存（1,1）
